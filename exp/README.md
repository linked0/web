# basic.py

## Setup
install matplotlib
```
pip install matplotlib
```

or install dependencies with requirements.txt
```
pip install -r requirements.txt
```

run basic.py
```
python basic.py
```

get dependency list
```
pip freeze > requirements.txt
```
