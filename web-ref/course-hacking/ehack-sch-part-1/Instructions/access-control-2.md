# Access Control Exercise 2

A new token "To The Moon" was deployed on Binance Smart Chain.
Unfortunately the deployer didn't mint any tokens for you, can you get some tokens anyway?

## Accounts
* 0 - Deployer
* 1 - User1
* 2 - Attacker (You)

## Tasks

### Task 1

### Task 2
